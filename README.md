# Ammar DAW-Based Portfolio

A personal portfolio website designed with a DAW-inspired interface. The idea behind this project was to make a portfolio that does not feel like a normal static developer website. Since I am a Computer Science student and also produce music, I wanted the design to combine both sides: software development and music production.

The website uses a dark studio-style layout with transport controls, track lanes, mixer-style skill cards, animated equalizer bars, waveform effects, and project sections arranged like a music production timeline.

## Overview

This portfolio presents my technical work, skills, achievements, and music links in a creative interface inspired by music production software such as FL Studio.

The main focus is my coding and software projects, while the music section adds a personal creative touch. The website includes an audio player layout with placeholder tracks that can be replaced with real MP3 or WAV files later.

## Features

- DAW-inspired user interface
- Dark aesthetic theme with neon-style accents
- Fixed top transport bar
- Play, pause, stop, next, and previous track controls
- Placeholder music player for MP3 and WAV files
- Animated waveform background
- Animated equalizer bars
- Mixer-style skills section
- Project cards displayed as DAW track lanes
- Expandable project details
- Achievements section
- Music portfolio section
- Contact section with links
- Responsive layout for desktop and mobile
- Smooth scrolling and hover animations
